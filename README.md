## 📋 Краткое описание сайта

Momentorest.ru — это сайт-визитка семейного ресторана **Momento**.
Сайт представляет ресторан, его меню, интерьеры залов, детскую комнату,
а также раздел «О нас» с описанием концепции заведения
Дизайн сайта выполнен в тёмных тонах с яркими акцентами,
содержит качественные фотографии еды, интерьера и гостей.
