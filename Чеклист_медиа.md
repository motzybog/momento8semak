# Чек-лист тестирования:  Корректное отображение изображений и прочего медиа-контента

## 1. Подготовительный этап

Тестовое окружение: Проверить отображение медиа-контента в нескольких браузерах (Chrome, Firefox, Safari, Edge) и на разных устройствах (десктоп, планшет, мобильный).

Тестовые сценарии: Подготовить проверку всех медиа-элементов сайта:

- [ ] Логотип в хедере.
- [ ] Hero-блок (фоновое изображение главной страницы).
- [ ] Фотогалерея на странице «О нас».
- [ ] Карточки меню на странице «Скачать меню».
- [ ] Карта на странице «Контакты».

---

## 2. Логотип

- [ ] Логотип «Momento» отображается чётко, без артефактов и пикселизации на всех страницах сайта.
- [ ] Логотип корректно отображается на светлом и тёмном фоне хедера (при смене фона при прокрутке — если предусмотрено дизайном).
- [ ] На мобильных устройствах логотип масштабируется корректно, без обрезания.

---

## 3. Главная страница — Hero-блок

- [ ] Фоновое изображение ресторана загружается полностью, без белых плашек и незагруженных областей.
- [ ] Изображение не обрезается по краям на широких экранах (1920px и выше).
- [ ] На мобильных устройствах фоновое изображение масштабируется корректно, ключевые элементы композиции (интерьер) остаются видимыми.
- [ ] Текст поверх изображения («"Momento" Family Restaurant», адрес, телефон, время работы) читаем и не сливается с фоном.
- [ ] Декоративные элементы (гирлянды, огни) на фото не создают артефактов при разных разрешениях экрана.

---

## 4. Страница «О нас» — Фотогалерея

### 4.1. Отображение изображений

- [ ] Все фотографии в галерее загружаются без ошибок (нет иконок сломанного изображения).
- [ ] Фотографии отображаются в правильных пропорциях, без искажений (растяжение, сжатие).
- [ ] Изображения не пикселизированы и имеют достаточное качество на экранах с высоким DPI (Retina).
- [ ] Сетка галереи отображается корректно: нет наложений фото друг на друга, нет неравномерных отступов.

### 4.2. Поведение галереи

- [ ] При клике на фотографию открывается увеличенный просмотр (lightbox) — если предусмотрено дизайном.
- [ ] В режиме lightbox изображение отображается чётко и по центру экрана.
- [ ] Навигация между фото в lightbox (стрелки «вперёд»/«назад») работает корректно.
- [ ] Закрытие lightbox (крестик или клик вне области) работает корректно.
- [ ] На мобильных устройствах галерея перестраивается в одно- или двухколоночный вид без обрезания фото.

---

## 5. Страница «Скачать меню» — Карточки меню

- [ ] Все 4 карточки содержат изображения: «Меню Italiano», «Винная карта», «Банкетное меню», «Детское меню».
- [ ] Изображения карточек загружаются без ошибок (нет иконок сломанного изображения).
- [ ] Изображения карточек отображаются в правильных пропорциях, без искажений.
- [ ] Текст на карточках (названия меню) читаем и не перекрывается элементами изображения.
- [ ] На мобильных устройствах карточки масштабируются корректно, изображения не обрезаются по значимым областям.

---

## 6. Страница «Контакты» — Карта

- [ ] Карта (Google Maps / Яндекс.Карты) загружается и отображается в нижней части страницы без ошибок.
- [ ] На карте корректно отмечена метка с адресом ресторана (проспект Ленина, 89а, Челябинск).
- [ ] Карта интерактивна: масштабирование (zoom in/out) работает корректно.
- [ ] Карта корректно отображается на мобильных устройствах, без выхода за границы блока.
- [ ] При медленном соединении карта показывает индикатор загрузки, а не пустой блок.

---

## 7. Общие проверки медиа-контента

- [ ] Ни на одной странице нет иконок сломанных изображений (alt-текст вместо картинки).
- [ ] Все изображения имеют заданный alt-текст (проверка через инструменты разработчика).
- [ ] При медленном интернет-соединении изображения загружаются постепенно (lazy loading), не блокируя отображение текстового контента.
- [ ] Нет изображений, выходящих за пределы своего контейнера и ломающих вёрстку страницы.
- [ ] На всех страницах отсутствуют видимые смещения контента в момент загрузки изображений (CLS — Cumulative Layout Shift).
